## Introduction
This folder just contains a random collection of helpful scripts.

- `.bash_profile_custom`
  A script which starts an `ssh-agent` as well as adds a cutom key during start up of a bash.
  The file should be sourced in the corresponding `.profile` file of a `bash`.
